# LAWA2 用户手册

> 版本：v3.8.0 | 更新日期：2026-06-18

---

## 欢迎使用 LAWA2

LAWA2 是一个养成式双语学习系统，通过日常场景沉浸 + 多 Agent 协作，帮助用户自然提升语言能力。

**核心理念**：LAWA2 不教语言，它让用户在两种语言里过生活。

---

## 快速开始

### 1. 注册/登录

1. 打开应用首页
2. 点击"开始"按钮
3. 输入用户名（无需密码）
4. 系统自动创建账户并登录

### 2. 设置用户画像

首次登录后，系统会引导你设置：

- **母语**：你的母语（默认中文）
- **目标语言**：你想学习的语言（默认英文）
- **当前水平**：beginner / intermediate / advanced
- **兴趣标签**：选择你感兴趣的话题

---

## 主要功能

### 🌿 习惯引擎（主 Agent）

**每日信息流**

系统每天推送三段内容：

- **晨间**（8:00）：资讯/新闻/知识
- **午间**（12:00）：趣闻/文化/生活
- **晚间**（20:00）：回顾/总结/反思

**桥梁对话**

与 AI 语伴进行中英双语对话：

1. 点击"桥梁"进入对话
2. 输入中文或英文消息
3. AI 自动用双语回复
4. 可以点赞、收藏、追问

**花园系统**

收集你学到的词汇和表达：

- 自动提取对话中的关键词
- 手动收藏感兴趣的表达
- 查看词汇成长曲线

### 📅 事项提醒 Agent

**节假日提醒**

- 自动推送中英双语节假日信息
- 包含文化背景介绍
- 可设置自定义提醒

**纪念日管理**

- 添加个人纪念日
- 设置提醒时间
- 自动生成祝福语

### 📸 拍照理解 Agent

**上传图片**

1. 点击"拍照"或上传图片
2. 支持 JPEG/PNG 格式
3. 最大 5MB

**AI 理解**

系统自动提供：

- 场景描述（中英双语）
- 关键词提取
- 相关表达推荐

**对话练习**

基于图片内容进行对话：

- 询问图片内容
- 学习相关词汇
- 分享对话到桥梁

### 👨‍💼 超级管理员（仅管理员）

**用户管理**

- 查看用户列表
- 搜索用户
- 禁用/启用账户
- 设置管理员权限

**内容管理**

- 管理种子语料
- 添加/编辑/删除内容
- 设置内容难度

**数据看板**

- 用户统计
- 行为分析
- 系统日志

## 使用技巧

### 提高学习效率

1. **每天三段**：按时阅读晨间/午间/晚间信息流
2. **主动对话**：多与桥梁对话，练习表达
3. **收藏词汇**：遇到不懂的词及时收藏
4. **拍照学习**：看到有趣的图片就上传学习

### 双语化使用

- 所有界面都是中英双语
- 默认显示双语，可点击展开翻译
- 错误提示、空状态都有双语说明

### 社交融入

- 使用社交场景语料预演真实对话
- 通过桥梁对话降低现实焦虑
- 获得"我被理解了"的正反馈

## 常见问题

### Q: 需要付费吗？

A: LAWA2 是完全免费的开源项目。

### Q: 支持哪些语言？

A: 目前支持中文 ↔ 英文双语。其他语言组合正在开发中。

### Q: 数据会丢失吗？

A: 所有数据存储在云端，不会丢失。

### Q: 可以离线使用吗？

A: 部分功能支持离线，但 AI 对话需要联网。

### Q: 如何反馈问题？

A: 可以在"设置"中提交反馈，或在 GitHub 上提交 Issue。

---

## 隐私与安全

- 所有数据加密存储
- 不收集个人敏感信息
- 用户可以随时删除账户
